What I'm learning (or have otherwise set my sights on)

  • Godot
    • Godot is a game engine used for making applications similar to the stuff in my "JavaScript" section. It uses a custom language adjacent to Python along with a node-based structure for projects, reminiscent to OOP.
    • Godot website
  • WebSockets
    • WebSockets is an API that allows real-time communication between two computers on the web. I'm interested in making my Canvas card games multiplayer, and this is the way to do that. If I ever manage to get this working, I will need to find a new means of deploying a website.
  • Django
    • Django is a Python library that is used to create a website based on a database. This allows for a more modular kind of site, where creating a new page doesn't require me to write an entire HTML document every time.
    • As said in the WebSockets section, I will need a new way of hosting this site if I manage to learn this technology and create something with it.
